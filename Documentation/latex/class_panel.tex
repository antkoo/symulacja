\doxysubsection{Detailed Description}
This class controls everything, that is happening within the simulation. As a JPanel, it draws all the buttons, the agents using {\ttfamily \doxylink{class_panel_a1f571ff47ec4c8ed1a889e405db76c8d}{Panel\#paint\+Component(\+Graphics)}} and all the counters {\ttfamily \doxylink{class_panel_afe3092d901bae52808250187ccb570fb}{Panel\#counter\+Updater(int)}}. It also controls the main {\ttfamily \doxylink{class_panel_a7dd188157b47c089996b769299bdcd37}{Panel\#resource\+Manager}} and a Swing timer\+: {\ttfamily \doxylink{class_panel_a78613c4bc7adca6d83ebb26241b73ff4}{Panel\#the\+Timer}} by re/starting the simulation using {\ttfamily \doxylink{class_panel_afcc925ac5975efcb4227c10f9932a96e}{Panel\#game\+Start()}} and updates every tick using {\ttfamily \doxylink{class_panel_affb317d6c209fc65598bd2841f3459ee}{Panel\#action\+Performed(\+Action\+Event)}}. It is also responsible for spawning zombies with {\ttfamily \doxylink{class_panel_ac63c9b28d4ccc66ffe26685341483e19}{Panel\#spawn\+Random\+Zombie()}} and plants with {\ttfamily \doxylink{class_panel_a49a142cef05292651f6b2bbb7f255c4c}{Panel\#spawn\+Random\+Plant()}}.
