\doxysubsection{Detailed Description}
A window, where a user selects the squares, where certain plants can spawn. The coordinate of square clicked is found using {\ttfamily \doxylink{}{Spawn\+Selector\#mouse\+Clicked(java.\+awt.\+event.\+Mouse\+Event)}} and the squares are stored in {\ttfamily \doxylink{class_spawn_selector_a97f84c9d40d350204beccc25e41d0efc}{Spawn\+Selector\#\+Spawn\+Squares}}. To choose a different plant, a {\ttfamily \doxylink{}{javax.\+swing.\+JRadio\+Button}} can be clicked, which is managed by {\ttfamily \doxylink{}{Spawn\+Selector\#action\+Performed(java.\+awt.\+event.\+Action\+Event)}}.
